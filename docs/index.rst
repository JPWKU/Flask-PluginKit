===============
Flask-PluginKit
===============

.. currentmodule:: flask_pluginkit

一个基于Flask的插件开发工具，支持多种扩展类型，您可以用来创建插件而无需更改核心代码。

插件可以是本地目录，也可以是第三方包（比如pypi）
